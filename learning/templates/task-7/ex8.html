Django Rest Framework (DRF) - это библиотека, которая работает со стандартными моделями Django для создания гибкого и мощного АРІ для проекта

Django CMS это современная платформа для веб-публикаций, построенная на Django, фреймворке веб-приложений «для перфекционистов с соблюдением сроков»

ASGI (Asynchronous Server Gateway Interface) является духовным наследником WSGI, предназначенным для обеспечения стандартного интерфейса между асинхронными веб-серверами, платформами и приложениями Python.

Python Social Auth это простой в настройке механигім социальной аутентификации/регистрации с поддержкой нескольких платформ и провайдеров аутентификации.

Django Crispy Forms. Лучший способ получить Django DRY формы. Создавайте программные макеты многократного использования из компонентов, имея полный контроль над отображаемым НТМІ. без записи НТМІ в шаблонах.
